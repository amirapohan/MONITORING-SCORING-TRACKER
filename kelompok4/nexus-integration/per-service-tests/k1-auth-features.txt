[5] PROFILE (butuh header Authorization: Bearer <accessToken>)

[6] GOOGLE OAUTH LOGIN — butuh idToken Google sah + env GOOGLE_CLIENT_ID.

    POST /auth/api/auth/google
    {
      "idToken": "<google-id-token-asli>"
    }

[7] INTERNAL API — header x-internal-api-key: <INTERNAL_API_KEY>
    (BUKAN JWT). Default contoh key "change-this-internal-api-key".
    Key salah -> 401/503 (test di-SKIP). INILAH yang dipakai K3 & K4.

    POST /auth/internal/validate-token  -> "Token is valid" (data.user = payload JWT)
    {
      "token": "<accessToken dari langkah [3]>"

    GET  /auth/internal/users/:id        (tanpa body) -> "User found"

    GET  /auth/internal/audit-logs       (tanpa body, query opsional) -> "Audit logs retrieved"

    GET  /auth/internal/talents/:talentId/project-completions (tanpa body)
